# Security Policy

## Reporting Security Issues

The kbind maintainers take security seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them through our [private vulnerability reporting](https://github.com/kbind-dev/kbind/security/advisories/new) form.

For more details, see the [kbind security policy](https://github.com/kbind-dev/kbind/blob/main/SECURITY.md).
